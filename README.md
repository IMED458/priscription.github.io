# Priscription Observation App

ვებ-აპი ინტენსიური თერაპიის დაკვირვების ფურცლისა და ექთნის ხარჯთაღრიცხვის ფორმებისთვის.

## სტრუქტურა

- `index.html` - მთავარი HTML (მარკაფი)
- `assets/css/app.css` - ყველა სტილი
- `assets/js/app.js` - მთელი ლოგიკა (Firebase, შაბლონები, ცხრილები, ბეჭდვა)
- `priscription.png` - favicon/logo

## გვერდები

- `1` - დაკვირვების ფურცელი (მედიკაცია)
- `2` - დაკვირვების ფურცელი (vitals/enteral/other)
- `ექთანი 1` - ხარჯთაღრიცხვა, ცარიელი ბადე (PDF-ს მსგავსი)
- `ექთანი 2` - ხარჯთაღრიცხვა, ჩამონათვალით (PDF-ს მსგავსი)

## შენიშვნები

- შაბლონის შენახვა/ჩატვირთვა მუშაობს Firebase-ზე.
- ბეჭდვა გათვლილია A4-ზე.
- `სინქრონიზაცია` ღილაკი ექთნის ფორმაზე ავსებს ისტორიის № და შემოსვლის თარიღს დაკვირვების ფურცლიდან.
